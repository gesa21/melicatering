# Images

Drop your photos into this folder using these exact filenames. The site already points at them.

| Filename | Where it shows | Suggested size |
| --- | --- | --- |
| `hero.jpg` | Home page hero, full width behind the headline | 1600 x 1000, landscape |
| `full-party.jpg` | Home card and The Full Party package | 1200 x 900 |
| `lunch-club.jpg` | Home card and the lunch club platters | 1200 x 900 |
| `savoury-table.jpg` | The Kids Table package | 1200 x 900 |
| `dolci-tray.jpg` | The Grown-Ups Table package | 1200 x 900 |
| `founders.jpg` | Founder section, square crop | 1000 x 1000 |

Tips for fast loading on phones:

- Save as JPG, quality around 80.
- Keep each file under 400 KB if you can.
- The hero is the most important one to compress well, since it loads first.

Until you add a photo, that spot shows a plain warm block, so the layout still looks tidy.
